Ready to go deeper?

Consider joining the WeSay Language Software Community site. Until we have real documentation, this is the place to go when something is unclear. As you ask questions, we'll answer them there. You can read old questions and answers there.

Please report bugs or request a feature using email to issues@wesay.org. Tell us that you tried it and what you did.  No, really. It's human nature to assume someone else had the same crash, but if it was easy to run into, we would have found it, and we would have fixed it. When you send to this address, your issue is automatically entered into our "todo list" and you will get updates as we work on the problem.

We are adding to WeSay's abilities regularly, and publishing new versions here.  Major new features are always announced on the WeSay news.
