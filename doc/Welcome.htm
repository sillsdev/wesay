Welcome to WeSay 1.5 Release

What's New in 1.5?

Version 1.5 mainly brings the functionality of the Windows version to the Linux platform. However, some improvements were made in the Windows version as well.

Common improvements

Improved setup of keyboards in the Configuration Tool.

Fix some issues with project creation.

Now possible to start a new project from Chorus Hub in the Configuration Tool.

Many bugfixes for Send/Receive with FLEx.

Improvements to the Windows version

Fixed some display issues in Dictionary Browse & Edit.

Installer now includes Gecko for rendering of complex scripts that require Graphite. It is off by default, but can be turned on. See How to Turn on GeckoFX for instructions.

New in the Linux version

Fix rendering issues for both roman and non-roman scripts (using GeckoFX). It is off by default, but can be turned on. See How to Turn on GeckoFX for instructions.

Enable use of ibus keyboards.

Enable automatic keyboard switching.

Fix several display issues (missing icons, clipped labels, etc.).

Fix the selection of semantic domains.

Enable add-ins.

Export to PDF and LibreOffice.

Enable several features that were new after WeSay 1.1:

  • Audio recording and playback
  • Chorus Notes
  • Art of Reading image collection
  • Send/Receive capability

Ready to go deeper?

Consider joining the Google Group. You can either use this group from your browser, through your email client, or a mix of both.  Until we have real documentation, this is the place to go when something is unclear.  As you ask questions, we'll answer them there.  A Google Group is a mailing list, but one that you can choose to have sent to your email box or wait for you to browse on the web, your choice. You can read old questions and answers.

Please report bugs or request a feature using email. Tell us that you tried it and what you did.  No, really.  It's human nature to assume someone else had the same crash, but if it was easy to run into, we would have found it, and we would have fixed it.    When you send to this address, your issue is automatically entered into our "todo list" and you will get updates as we work on the problem.

We are adding to WeSay's abilities regularly, and publishing new versions here.  Major new features are always announced on the blog.
