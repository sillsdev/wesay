About this version

WeSay comes in two flavors: 'stable' and 'beta'.   Once a version is 'stable", it will only receive critical bug fixes, but not new features or anything risky.

If crashes or other bugs would be a major problem for you, please uninstall this version and get our most recent stable one (one without the 'beta' tag).  If you are helping us advance WeSay, are evaluating WeSay's capabilities, or are just willing to risk the occasional bug, this is the version for you.  This is where we are currently adding new abilities in response to your requests, and we need people to help ensure that it can become stable, too. If you report a bug via issues@wesay.org, we can often times fix it and provide a new release within hours.  So let us know if you're in a hurry for a fix.
