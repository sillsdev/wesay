Welcome to WeSay

You have installed an early test version of WeSay.  This is a team effort, and we really appreciate your willingness to give WeSay a spin.

Please, do not entrust data to this version.  This is not a released product.  It is not a beta.  It’s not even an alpha.  If you are putting in any data in that you don’t want to lose, please make sure to backup your project folder constantly. (BTW: even when WeSay it crashes, you are unlikely to loose data, even your previous keystroke).

What Now?

·       Get your head around WeSay… it’s really different.  It’s aimed at a very different user than you are, and approaches the task of dictionary making in a new, hyper-task-oriented way.  Using the provided samples, play around with both WeSay and the WeSay Setup application.  To do this, go to your Windows Start menu and find “WeSay”.  Choose one of the icons there which open the sample project.

·       Have existing data you want to load in? 

You’d need to get your data into the LIFT XML format, and the tools for doing that are in their infancy.  If you happen to be using the SIL Southeast Asia SFM standard, there is an upgrade to you Toolbox type files which will let you round-trip your data from LIFTßà SEADs.  If you’re using FLEx, you can tell us “hey, update the LIFT export to the current version, already”.  Otherwise, write to us to develop a plan for you and your data.
